## SEO Best Practices

1. **First 150 characters** are visible before "Show More" - make them count!
2. Include **primary keyword** in first sentence
3. Use **exact match keywords** from title in description
4. Add **timestamps** for YouTube chapter feature
5. Include **brand mentions** (RideWire, COCO) early
6. Use **question format** to match search queries
7. Add **location keywords** if relevant (e.g., "California emissions")
